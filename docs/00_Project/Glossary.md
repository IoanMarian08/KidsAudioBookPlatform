## Table of Contents


This glossary defines project terminology used across product, design, architecture and engineering documents.

## API Gateway

Single entry point routing client requests to backend microservices.

## Account

A registered parent identity used to access the platform.

## Admin Dashboard

Internal web application used to manage content, users, offers and platform configuration.

## Advertisement

A promotional unit shown only to Free users under strict child-safe rules.

## Ambient Sound

Background sound such as rain, waves or white noise played in the app or during a story.

## App Store

Apple distribution platform for iOS applications.

## Audit Log

Persistent record of important actions, usually for security or compliance.

## Author

Future role that may create and submit stories.

## Backend

Server-side system supporting the mobile application and admin dashboard.

## Biometric Authentication

Device-level authentication such as Face ID or fingerprint.

## CDN

Content Delivery Network used to deliver media quickly.

## Category

Permanent classification of stories, such as Animals or Bedtime.

## Child Experience

The safe and simplified area used by children.

## Child Lock

Mechanism preventing children from accessing sensitive actions.

## Child Profile

Personalized profile representing one child.

## Child Room

Main personalized child screen replacing the concept of Home.

## Collection

Temporary or curated grouping of content.

## Content Service

Backend service managing stories, series, episodes and categories.

## Correlation ID

Identifier used to trace a request across multiple services.

## Docker

Containerization platform used for local and production environments.

## Episode

A story that belongs to a series.

## Flutter

Framework used to build the mobile app for Android and iOS.

## Free Plan

Limited plan with free stories and ads.

## Illustration

Image associated with a story or story segment.

## JWT

Token format used for authentication.

## JavaDoc

Documentation format for Java classes and methods.

## MVP

Minimum Viable Product.

## Media Asset

Audio, image or other file managed by the media service.

## Microservice

Independent backend service with a focused responsibility.

## Notification

Message sent or displayed to a user and persisted in database.

## Parent Account

Adult account that owns child profiles.

## Parent Zone

Protected area for account, subscription and settings.

## Premium

Paid plan unlocking full features.

## RabbitMQ

Message broker for asynchronous communication.

## Redis

In-memory store used for caching and rate limiting.

## Series

Group of related episodes.

## Story

Primary content unit containing audio, text and illustrations.

## Synchronized Text

Text highlighted according to audio timing.

## Trial

Temporary free Premium access.

## White Noise

Constant ambient sound used for relaxation or sleep.
