# ADR-003: Use PostgreSQL as Primary Database

| Field | Value |
|---|---|
| Status | Accepted |
| Date | 2026-07-04 |
| Owner | Ioan Marghioala |
| Related Area | Foundation Architecture |

---

## Context

The platform has structured data: users, child profiles, stories, subscriptions, notifications, audit logs and content metadata.

---

## Decision

Use PostgreSQL as the primary relational database.

---

## Alternatives Considered

- MySQL
- MongoDB
- PostgreSQL
- Firebase

---

## Consequences

- Strong relational consistency
- Good indexing
- Mature ecosystem
- Works well with Spring Data JPA

---

## Notes

This decision may be revisited if product scale, team size or business constraints change.
