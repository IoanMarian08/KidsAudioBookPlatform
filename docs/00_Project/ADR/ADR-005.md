# ADR-005: Use Docker-First Development

| Field | Value |
|---|---|
| Status | Accepted |
| Date | 2026-07-04 |
| Owner | Ioan Marghioala |
| Related Area | Foundation Architecture |

---

## Context

The project depends on PostgreSQL, Redis, RabbitMQ and multiple backend services. Installing each dependency manually on Windows would create inconsistent environments.

---

## Decision

Use Docker and Docker Compose as the standard local infrastructure strategy.

---

## Alternatives Considered

- Local native installs
- Docker Compose
- Cloud-only development

---

## Consequences

- Repeatable setup
- Cleaner local environment
- Closer to production
- Requires Docker knowledge

---

## Notes

This decision may be revisited if product scale, team size or business constraints change.
